# Проект: Интернет-магазин кроссовок - React Sneakers

## Используемые техники и технологии
- ReactJS  + хуки
- React Router
- React Context
- React Skeleton
- react-slick
- JSX
- Java Script
- html
- SCSS / CSS Modules
- Axios
- адаптивная верстка (Flexbox)
- Библиотека npm с готовыми классами macro-css
